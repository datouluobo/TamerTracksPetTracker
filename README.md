# TamerTracksPetTracker

![Logo](pics/logo_1024.png)

**TamerTracksPetTracker** 是一款专为《魔兽世界》猎人玩家设计的专业插件，专注于**潘达利亚稀有隐藏宠物（需追踪足迹捕捉）**的足迹追踪、自动报警与航线测绘。

通过本插件，你可以轻松记录下稀有宠物的出没规律，并在发现足迹时通过声光报警第一时间锁定目标。

---

## 🌟 核心功能

*   **🔍 足迹悬停报警**：当鼠标悬停在地面隐藏足迹（Tracks/Prints）上时，插件会立即弹出**全屏警报**并播放提示音，提醒你发现了目标。
*   **📡 自动报警系统**：支持自定义警报声效与团队警报文字（Raid Warning），确保你在长时间挂机蹲守时不会错过目标。
*   **🗺️ 实地测绘与路线记录**：一键记录当前坐标，自动在地图上生成宠物的巡逻轨迹图，帮你直观掌握宠物的活动范围。
*   **⚙️ 实时 HUD 列表**：独立的测绘清单界面，方便对已记录的数据进行增删改查。
*   **🔗 跨服数据分享**：内置 P2P 通信协议，支持在频道内点击链接，直接向其他玩家同步完整的宠物航线数据。

---

## 🛠️ 使用方法

### 1. 聊天命令 (Slash Commands)
*   **/tr** - 开启/关闭测绘 HUD 主界面。
*   **/tra** - **[录入]** 记录你当前所在的坐标点到当前选择的宠物航线中。

### 2. 追踪足迹
*   在潘达利亚地图寻找那些只有猎人能看见的“泥泞的脚印”。
*   当你的鼠标划过脚印，插件会发出提示音并显示红色警报。
*   使用 **/tra** 将这个脚印的位置记录下来，多次记录后，你就能在地图上清晰地看到该稀有宠物的巡逻路径。

### 3. 数据同步与分享
*   在 HUD 界面可以生成分享链接发送到频道。
*   其他玩家点击链接后，会通过隐藏频道自动向你请求完整数据并合并到他们的本地数据库。

## 🤝 数据共建说明 (重要！)

本插件的强大之处在于**数据的积累**。目前内置数据仅为初始参考，完美的稀有宠物路径需要广大玩家不断在实地进行“测绘”完善。

### 如何贡献数据？
我们的目标是建立一个最全的潘达利亚宠物坐标库。如果你测绘出了完整的宠物路径，欢迎通过以下方式分享：

1.  **GitHub 提交**：
    将你插件配置目录下的 `WTF\Account\你的账号\SavedVariables\TamerTracksPetTracker.lua` 文件中的内容，作为 Issue 或 Pull Request 提交给我们。
2.  **社区分享**：
    在公频或其他平台分享你的测绘成果，让更多人受益。

> **推荐大家多多分享！** 只有每个人都贡献出一小段路径，我们才能拼凑出艾泽拉斯最完整的猎手地图。

## 📦 安装说明

1.  下载本项目压缩包。
2.  解压后将 `TamerTracksPetTracker` 文件夹放入你的魔兽世界目录：`World of Warcraft\_classic_\Interface\AddOns\`。
3.  重启游戏即可使用。
